# enerji-anomali-risk-dashboard
Power BI dashboard for electricity consumption anomaly detection and operational risk analysis

# Elektrik Tüketimi Anomalileri – Operasyonel Risk Dashboard

Bu proje, elektrik tüketim verilerinde anomali tespiti ve operasyonel risklerin
yönetici seviyesinde izlenmesi amacıyla Power BI kullanılarak geliştirilmiştir.

## Dashboard İçeriği
- Toplam Anomali ve Riskli Tesisat KPI’ları
- Anomali Türü Dağılımı
- Günlük Anomali Trendi
- Bölgesel Anomali Yoğunluğu (Harita)
- Öncelikli Müdahale Gereken Tesisatlar (Top 10)

## Dosyalar
- `enerji-anomali-dashboard.pbix` — Power BI raporu

## Kullanılan Araçlar
- Power BI
- DAX
- Zaman serisi analizi
- Coğrafi görselleştirme

## Kullanım
- `enerji-anomali-dashboard....pbix` dosyasını indirin
- Windows Power BI Desktop ile açın
